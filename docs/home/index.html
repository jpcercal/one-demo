# Home

The first page we are going to cover is the homepage.

It's an important part of the process as it will set the expectation with your visitor.

# Changing the hero

The easiest way to get it changed would be by replacing the image living on images/one/home/hero.png by a new one of your own preference. The image must have a resolution of 555x555px.

If you want, you can also change the image path directly into the HTML code, if this is your case please open the template html file on the root folder and search for the block shown below:

<!-- ==========| Hero #-->
<div class="col-lg-7 pr-5">
  <div class="hero-bg"><img class="w-100" alt="One" src="images/one/home/hero.png"></div>
</div>

You have to change the line: src="images/one/home/hero.png" pointing it out to a different image.

# Changing the Sentences

You can change the sentences listed on the home page by simply changing the following lines of the template:

<!-- ==========| Heading #-->
<h2 class="h4 text-center text-lg-left d-inline mr-3" id="typed">Little things make big days.</h2>
<div class="d-none" id="typed-strings">
  <p>Little things make big days.</p>
  <p>Dream it. Wish it. Do it.</p>
  <p>Keep going. You can do it!</p>
</div>
<p class="my-5 h1 text-center text-lg-left">Donâ€™t wait for opportunity. Create it.</p>

Because of a plugin, the sentences listed inside of <div class="d-none" id="typed-strings"> will be typed in like a robot on the fly (when you render the page on your browser). There's nothing required to do as this works out of the box.
