# Changelog

# Version 1.0.0

  • Template created with one theme ready to be used
  • Multi-purpose themes are possible given the dynamic structure
  • Introduced a basic documentation on how to customize the template
  • Images optimized out of the box to the web
  • HTML document validated by W3C as a valid HTML5 document
  • Validation of compatibility with desktop browsers on Windows 10 Pro (Edge, Chrome, and Firefox) and MacOS Catalina (Safari, Chrome, and Firefox)
  • Validation of compatibility with mobile devices Android 9 (Google Chrome) and iOS 12+ (Safari)
  • The theme uses the following external dependencies:
    • Twitter Bootstrap 4.4.1
    • jQuery 3.3.1
    • Fancybox 3.5.7
    • jQuery One-Page Navigation 3.0.0
    • AOS 2.3.4
    • Animate+ 2.1.1
    • Typed.js 2.0.11
    • Font-awesome 5.13.0
    • pe7-icon 1.0.4
    • Web Fonts Noto Sans JP & Raleway
    • External images (a few of them - like the homepage hero - are allowed to use only with copyright attribution)
