# About

The about page.

# Changing the title

You can change the title by changing the block listed below:

<!-- ==========| About [section] #-->
<section class="section section--secondary" id="about">
  <div class="container">

    <!-- ==========| Header #-->
    <div class="row no-gutters">
      <div class="col-md-12 py-4 text-center">
        <h2 class="h2 text-white">One Solution for your Entire Business</h2>
        <p class="mt-3 mb-5 text-white">Now there is a new integrated end-to-end solution for all processes.</p>
      </div>
    </div>

# Changing the features and the icons

You can change the features by changing the block(s) shown below:

<!-- ==========| Features #-->
<div class="row">
  <div class="col-md-6 ml-md-auto" data-aos="fade-up">
    <div class="card mb-4 p-1 card--border-bottom">
      <div class="row no-gutters pl-4 pr-4">
        <div class="col-auto"><i class="pe-7s-medal"></i></div>
        <div class="col px-2">
          <div class="card-body">
            <h2 class="card-title h4">Premium Features</h2>
            <p class="card-text mt-3">Lorem ipsum dolor sit amet, consectetur adipiscing elit. Mauris cursus laoreet augue, a accumsan nulla egestas facilisis. Curabitur quam ex, pharetra sed est vitae, euismod interdum dolor. Praesent.</p>
          </div>
        </div>
      </div>
    </div>
  </div>
  
  // ...

The icon could be changed by replacing the class value <i class="pe-7s-medal"></i>. A list of all possible values can be found on https://themes-pixeden.com/font-demos/7-stroke/index.html
