# Introduction

Thank you very much for acquiring our solution. This guide will show you how to get started with this template and how you could be customizing it as you go on.

It's not required to have any knowledge about HTML or any other technology to do so, but if you have a hard time trying to setup anything, get in touch with us, and we can find a way to improve this document.

During the next pages you will see section by section of the template and how to get it changed.

Get Started
